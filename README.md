# Telegram slot watcher bot

Бот следит за Telegram-стикером игрового автомата `🎰`, отвечает на нужные комбинации и ведет статистику по чату и пользователям.

## Подготовка

1. Создайте бота через [@BotFather](https://t.me/BotFather) и получите token.
2. Скопируйте `.env.example` в `.env`.
3. Вставьте token в `TELEGRAM_BOT_TOKEN`.
4. Настройте:
   - `ALLOWED_CHAT_IDS` - ID чатов, где бот имеет право работать.
   - `OWNER_USER_IDS` - ваш Telegram user ID для `/chatid`, `/resetstats` и поиска подарков owner при `777`.
   - `SMALL_GIFTS` - призы для трех BAR, трех лимонов или трех вишен/винограда.

## Запуск

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python bot.py
```

## Команды

```text
/chatid - показать ID текущего чата и ваш user ID
/stats - показать статистику по разрешенному чату
/resetstats - обнулить статистику текущего чата, только для OWNER_USER_IDS
/emojiid - показать custom_emoji_id из сообщения, на которое ответили командой
/settext - сохранить шаблон сообщения в личке с ботом
/texts - показать сохраненные шаблоны в личке с ботом
```

Owner из `OWNER_USER_IDS` получает личные уведомления о выигранных подарках. Для этого владелец должен сначала открыть бота в Telegram и нажать `Start`, иначе Telegram не разрешит боту писать ему в личку.

## Шаблоны сообщений

Команда `/settext` работает только в личке с ботом и только для `OWNER_USER_IDS`.

```text
/settext 777 username выбил 777 и забирает nft_url
/settext 77x username выбил 77X. Срочно додэп.
/settext triple username выбил combination и получает giftr
```

Для длинного текста или Telegram custom emoji отправьте боту отдельное сообщение с нужным оформлением, затем ответьте на него:

```text
/settext 777
```

Доступные placeholders:

```text
username - username или имя победителя
nft_url - случайная ссылка на gift с профиля owner
giftr - роза, мишка, сердце или подарок
combination - выпавшая комбинация
```

## Защита

Бот обрабатывает спины и команду `/stats` только в чатах из `ALLOWED_CHAT_IDS`.

Чтобы узнать ID чата:

1. В `.env` временно укажите свой ID в `OWNER_USER_IDS`.
2. Запустите бота.
3. Напишите `/chatid` в нужном чате.
4. Скопируйте `ID этого чата` в `ALLOWED_CHAT_IDS`.
5. Перезапустите бота.

Пример:

```env
OWNER_USER_IDS=123456789
ALLOWED_CHAT_IDS=-1001234567890
```

Если разрешенных чатов несколько:

```env
ALLOWED_CHAT_IDS=-1001111111111,-1002222222222
```

Если `ALLOWED_CHAT_IDS` пустой, бот не будет считать спины и показывать статистику.

## Важно для групповых чатов

Если бот должен видеть все сообщения в группе, отключите privacy mode у бота:

1. Откройте [@BotFather](https://t.me/BotFather).
2. Выполните `/setprivacy`.
3. Выберите своего бота.
4. Выберите `Disable`.

После этого добавьте бота в нужный чат.

## Как это работает

Telegram отправляет игровой автомат как `dice`-сообщение. Для emoji `🎰` комбинации приходят числом от `1` до `64`.

```text
1  = BAR BAR BAR
22 = вишни/виноград вишни/виноград вишни/виноград
43 = лимон лимон лимон
16, 32, 48 = 7 7 любой другой символ
64 = 7 7 7
```

Статистика хранится в SQLite-файле `bot_stats.sqlite3`.
