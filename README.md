# Telegram slot watcher bot

Бот следит за Telegram-стикером игрового автомата `🎰`, отвечает на нужные комбинации и ведет статистику по чату и пользователям.

## Подготовка

1. Создайте бота через [@BotFather](https://t.me/BotFather) и получите token.
2. Скопируйте `.env.example` в `.env`.
3. Вставьте token в `TELEGRAM_BOT_TOKEN`.
4. Настройте:
   - `ALLOWED_CHAT_IDS` - ID чатов, где бот имеет право работать.
   - `OWNER_USER_IDS` - ваш Telegram user ID для `/chatid`, `/resetstats` и поиска подарков owner при `777`.
   - `SMALL_GIFTS` - призы для трех BAR, трех лимонов или трех вишен/винограда.

## Запуск

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python bot.py
```

## Команды

```text
/chatid - показать ID текущего чата и ваш user ID
/userid - показать user ID; если ответить на сообщение пользователя, покажет его ID
/stats - показать общую статистику по разрешенному чату
/mystats - показать личную статистику; в группе можно ответить на сообщение пользователя, в личке бот предложит выбрать чат кнопкой
/dailybonus - получить ежедневный бонус TEM в личке с ботом
/help - показать помощь для обычного пользователя
/resetstats - обнулить статистику текущего чата, только для OWNER_USER_IDS
/resetuserstats - обнулить статистику конкретного пользователя в текущем чате, только для OWNER_USER_IDS
/emojiid - показать custom_emoji_id из сообщения, на которое ответили командой
/settext - сохранить шаблон сообщения в личке с ботом
/sethelp - настроить текст /help в личке с ботом, только для OWNER_USER_IDS
/setranks - настроить список рангов в личке с ботом
/setchance - настроить проценты и частоту chance-сообщения
/texts - показать сохраненные шаблоны и настройки кнопками в личке с ботом
```

Owner из `OWNER_USER_IDS` получает личные уведомления о выигранных подарках. Для этого владелец должен сначала открыть бота в Telegram и нажать `Start`, иначе Telegram не разрешит боту писать ему в личку.

## Шаблоны сообщений

Команда `/settext` работает только в личке с ботом и только для `OWNER_USER_IDS`.

```text
/settext 777 username выбил 777 и забирает nft_url
/settext 77x username выбил 77X. Срочно додэп.
/settext triple username выбил combination и получает giftr
/settext stats Всего спинов: total_spins. 777: jackpots. 77X: two_sevens. Топ: top5
/settext mystats username: total_spins спинов, ранг rank, 777 jackpots, 77X two_sevens, три в ряд three_in_row
/settext welcome Привет, username. Добро пожаловать в chat.
/settext milestone username сделал milestone спинов. 777: jackpots, 77X: two_sevens, три в ряд: three_in_row
/settext dailybonus username получил daily_bonus TEM. Баланс: balance TEM
/settext dailyreminder username, ежедневная награда доступна. Напиши /dailybonus
/settext chance username, шанс на 777 примерно chance
```

Chance-сообщение настраивается отдельно:

```text
/setchance perc10 spins5
```

`perc10` умножит показываемый шанс на 10, `perc2` - на 2. `spins5` означает, что сообщение будет отправляться примерно раз в 5 спинов.

Команда `/texts` показывает таблицу команд установки и кнопки. Нажмите кнопку нужной настройки, чтобы увидеть, какой текст сейчас стоит.

Help-сообщение для обычных пользователей настраивается отдельно:

```text
/sethelp Привет, username. Баланс: balance TEM. Ранг: rank
```

Для Telegram custom emoji отправьте боту готовое сообщение с оформлением и ответьте на него `/sethelp`.

Шаблон `welcome` используется и для `/start` в личке, и для приветствия новых участников в разрешенной группе.
Шаблон `milestone` отправляется в чат на каждом кратном 25 спине пользователя: 25, 50, 75, 100 и дальше.
Общая статистика отправляется автоматически каждый час во все разрешенные чаты и дополнительно при каждом 100-м спине чата.
Ранг пользователя повышается каждые 100 спинов. До 100 спинов это ранг `новичок`, после 100 - следующий уровень, после 200 - еще следующий.
Список рангов настраивается owner-командой `/setranks` в личке с ботом.
За каждый спин бот начисляет TEM: обычный спин +1, 77X +6 всего, три в ряд +11 всего, 777 +26 всего.
`/dailybonus` можно забрать один раз в день в личке с ботом. Шанс: 1-5 TEM - 80%, 10 TEM - 15%, 25 TEM - 5%.
Пользователям, которые запускали бота в личке, бот раз в день пишет напоминание забрать `/dailybonus`.
Примерно раз в 5 спинов бот пишет подсказку с примерным шансом выбить 777.

Индивидуальный шаблон личной статистики для конкретного пользователя задается отдельно и не меняется, когда вы меняете общий `/settext mystats`.

```text
/setusertext USER_ID mystats username, твоя личная статистика: total_spins спинов
```

Ранги задаются через запятую. Первый ранг используется с 0 спинов, второй с 100, третий с 200 и дальше:

```text
/setranks новичок, искатель, крутящий, азартный, мастер, легенда
```

Telegram custom emoji в рангах тоже поддерживаются. Если список длинный, можно отправить боту сообщение с рангами и ответить на него командой `/setranks`.

Чтобы узнать `USER_ID`, ответьте в чате на сообщение пользователя:

```text
/userid
```

Для длинного текста или Telegram custom emoji отправьте боту отдельное сообщение с нужным оформлением, затем ответьте на него:

```text
/settext 777
```

Доступные placeholders:

```text
username - username или имя победителя
nft_url - случайная ссылка на gift с профиля owner
giftr - роза, мишка, сердце или подарок
combination - выпавшая комбинация
chat - название чата
total_spins - всего спинов
jackpots - количество 777
two_sevens - количество 77X
three_bars - три BAR
three_grapes - три винограда
three_lemons - три лимона
three_in_row - всего три в ряд
other_spins - остальные спины
top5 - топ 5 по количеству спинов
luckiest_777 - самый везучий по количеству 777
luckiest_ratio - самый везучий по проценту 777 от всех спинов
milestone - текущий рубеж спинов
rank - текущий ранг пользователя
balance / tem_balance - баланс TEM
daily_bonus / bonus - ежедневный бонус TEM
chance / chance_percent - примерный шанс 777
```

## Защита

Бот обрабатывает спины и команду `/stats` только в чатах из `ALLOWED_CHAT_IDS`.
Пересланные сообщения со слотами не засчитываются, чтобы нельзя было повторно начислять jackpot или статистику форвардом.

Чтобы узнать ID чата:

1. В `.env` временно укажите свой ID в `OWNER_USER_IDS`.
2. Запустите бота.
3. Напишите `/chatid` в нужном чате.
4. Скопируйте `ID этого чата` в `ALLOWED_CHAT_IDS`.
5. Перезапустите бота.

Пример:

```env
OWNER_USER_IDS=123456789
ALLOWED_CHAT_IDS=-1001234567890
```

Если разрешенных чатов несколько:

```env
ALLOWED_CHAT_IDS=-1001111111111,-1002222222222
```

Если `ALLOWED_CHAT_IDS` пустой, бот не будет считать спины и показывать статистику.

## Важно для групповых чатов

Если бот должен видеть все сообщения в группе, отключите privacy mode у бота:

1. Откройте [@BotFather](https://t.me/BotFather).
2. Выполните `/setprivacy`.
3. Выберите своего бота.
4. Выберите `Disable`.

После этого добавьте бота в нужный чат.

## Как это работает

Telegram отправляет игровой автомат как `dice`-сообщение. Для emoji `🎰` комбинации приходят числом от `1` до `64`.

```text
1  = BAR BAR BAR
22 = вишни/виноград вишни/виноград вишни/виноград
43 = лимон лимон лимон
16, 32, 48 = 7 7 любой другой символ
64 = 7 7 7
```

Статистика хранится в SQLite-файле `bot_stats.sqlite3`.
